.. _swift:

^^^^^^^^^
swift CLI
^^^^^^^^^
The OpenStack tool primarily used for
managing Cloud Files is written
in Python and called *swift*.
It is also known as
*python-swiftclient*.

The following OpenStack documents can help you install swiftclient
and learn to use it:

* :os-docs:`Install the OpenStack command-line clients <user-guide/common/cli_install_openstack_command_line_clients.html>`

* :os-docs:`OpenStack Command-Line Interface Reference <cli-reference/content/index.html>`,
  especially at
  :os-docs:`Object Storage command-line client <cli-reference/content/swiftclient_commands.html>`

* :os-docs:`swiftclient web <developer/python-swiftclient>`

* :os-wiki:`Configuring swift <wiki/ConfigureSwift>`
