~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Networking considerations for OnMetal servers
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
OnMetal servers are provisioned with redundant 10 GigE connections,
which enables them to run high-bandwidth applications without data throughput
issues.

Unlike virtual servers, OnMetal servers cannot add or remove networks.
