v2.23.0, July 8, 2015   
---------------------

What's new
~~~~~~~~~~

-  Incremental backups are no longer labeled as *incremental*.

-  A 1-to-1 relationship between a schedule and an instance has been
   established.

-  When you specify that a :code:`full day!=today`, no incremental backup is
   performed until after the full date.
